When you need to download large files, like Linux distributions, it's wise not to fully trust the browser for these downloads, since some packets can be lost without the download throwing an error, and then we have problems with the system without knowing why.

So, to practice building bash scripts and also to automate this small task, I created wgetmd5, which is nothing more than the combination of wget and md5sum to calculate the md5 hash of the download and compare it with the md5 file given as a parameter.
