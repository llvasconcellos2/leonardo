In this post we'll detail the installation and configuration of SafeStrap Recovery to create a DualBoot between the Stock ROM — which supports HDMI and FullHD (1080p) video recording — and CyanogenMod 11 with the latest version of Android, 4.4 KitKat.

The idea is to use CyanogenMod daily to enjoy KitKat's features while keeping the Stock ROM with Jelly Bean for use connected to a monitor.

For that you'll need to download the following files:

*   [SafeStrap](http://www.4shared.com/mobile/J74W4X6Dba/Safestrap-UMTS_Spyder-365.html "SafeStrap 3.65 Recovery for Droid Razr") 3.65 for umts\_spyder (codename of the Motorola Razr GSM)
*   [CyanogenMod](http://download.cyanogenmod.org/?device=umts_spyder "CyanogenMod umts_spyder Droid Razr GSM") (I also saved the version I used here: [cm-11-20140104-SNAPSHOT-M2-umts\_spyder.zip](http://www.4shared.com/zip/u4qpDocNce/cm-11-20140104-SNAPSHOT-M2-umt.html "cm-11-20140104-SNAPSHOT-M2-umts_spyder.zip"))
*   [Google Apps](http://wiki.cyanogenmod.org/w/Google_Apps "Google Apps CyanogenMod") (Google apps that aren't included in CyanogenMod by default). I also saved the version I used here: [gapps-kk-20131208.zip](http://www.4shared.com/zip/O7qbAezgba/gapps-kk-20131208.html "Google Apps KitKat 08/12/2013"))

Keep in mind that you need root on your Razr in order to install Safestrap. If you don't have it yet, follow the tutorial below:

Follow these steps:

1.  Copy the CyanogenMod and Google Apps zip files to the device.
2.  Copy the Safestrap-UMTS\_Spyder-3.65.apk file to the phone and install it.
3.  Open the app and grant root permission.
4.  Select the option: Install Recovery
5.  Turn the device off and back on.
6.  The SafeStrap Boot menu will appear as shown in the image:
    
    [![Safestrap menu options](/old-blog/uploads/2014/01/IMG_0122.jpg)](/old-blog/uploads/2014/01/IMG_0122.jpg)
    
    SafeStrap Boot Options
    
7.  Select \[MENU\] to enter Safestrap. In the header the message **ROM Slot: stock** will appear in red, which means the changes will be made to the factory ROM, so go to **Boot Options** and select **ROM-Slot-1**
    
    [![Safestrap Boot Options](/old-blog/uploads/2014/01/IMG_0133.jpg)](/old-blog/uploads/2014/01/IMG_0133.jpg)
    
    Safestrap Boot Options
    
8.  The next screen shows the size options for the new system image where CyanogenMod will be installed. Leave **System partition size** at 700 MB and **Cache partition size** at 300 MB (default values). The **Data partition size** option is the space reserved for installing apps. The first time I created the image I lowered the Data partition to 1000 MB, but the space soon filled up and I had problems, so if you use a lot of apps leave it at 2000 MB or increase it up to 4000 MB. Remember that this space will be taken from your Razr's internal storage.
    
    [![Creating a ROM Slot in SafeStrap](/old-blog/uploads/2014/01/IMG_0135.jpg)](/old-blog/uploads/2014/01/IMG_0135.jpg)
    
    Creating a ROM Slot in SafeStrap
    
9.  The ROM slot will be created:
    
    [![Creating ROM Slot](/old-blog/uploads/2014/01/IMG_0139.jpg)](/old-blog/uploads/2014/01/IMG_0139.jpg)
    
    Creating ROM Slot
    
10.  After the process finishes, check whether the SafeStrap header shows the message **ROM Slot: rom-slot1**. If the Stock ROM is still selected, go to **Boot Options – ROM-Slot-1 – Activate**.
     
     [![ROM Slot 1 active in SafeStrap](/old-blog/uploads/2014/01/IMG_0142.jpg)](/old-blog/uploads/2014/01/IMG_0142.jpg)
     
     ROM Slot 1 active in SafeStrap
     
11.  In the SafeStrap main menu, select Install.
     
     [![SafeStrap main menu](/old-blog/uploads/2014/01/IMG_0151.jpg)](/old-blog/uploads/2014/01/IMG_0151.jpg)
     
     SafeStrap main menu
     
12.  A screen will appear for selecting the zip file to be installed. Navigate to the directory where you copied the zip files and first select the CyanogenMod file (cm-11-xxxx.zip), then go to Add More Zips and select the Google Apps zip file (gapps-xxx.zip).
     
     [![Selecting a Zip to install in SafeStrap](/old-blog/uploads/2014/01/IMG_0153.jpg)](/old-blog/uploads/2014/01/IMG_0153.jpg)
     
     Selecting a Zip to install in SafeStrap
     
13.  Swipe to confirm the installation (**Swipe to confirm flash**). On my device the GApps installation fails, but I ignored the message, and when booting the ROM and using Google Play for the first time an error occurs due to a missing component that I believe Play installs automatically, since the message didn't repeat and everything worked fine. This probably happens because, as of now, CM11 on the Razr isn't stable and is still in development. If anyone runs into a bug, please share it here in the comments so we can stay informed. 🙂
     
     [![Swipe to confirm the installation (SafeStrap)](/old-blog/uploads/2014/01/IMG_0156.jpg)](/old-blog/uploads/2014/01/IMG_0156.jpg)
     
     Swipe to confirm the installation
     
14.  The installation process will start:
     
     [![Installing zips using SafeStrap](/old-blog/uploads/2014/01/IMG_0161.jpg)](/old-blog/uploads/2014/01/IMG_0161.jpg)
     
     Installing zips using SafeStrap
     
15.  When the installation finishes, select the **Wipe cache/dalvik** option, then go to the main menu and select **Reboot – System** to reboot into your new Android 4.4 KitKat!
     
     [![CyanogenMod 11 KitKat boot animation](/old-blog/uploads/2014/01/IMG_0167.jpg)](/old-blog/uploads/2014/01/IMG_0167.jpg)
     
     CyanogenMod 11 KitKat boot animation
     

Afterward you can switch between the active ROM Slots to choose which system to boot.
