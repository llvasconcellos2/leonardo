In this post we'll show step by step the installation of Linux Mint Debian Edition, which is a bit simpler than installing Debian Testing "Wheezy", the distribution it's based on. To complete this tutorial you'll need internet access, a blank or rewritable CD or a pendrive, and a little patience.

So, you who want to set Windows aside and use a real operating system, or you who already use Linux but are fed up with Ubuntu, how about some Mint Linux?

Shall we?

Installing Linux Mint Debian Edition 201204

  
The first step is to download the ISO image of the LMDE installation CD. To do this, go to the [www.linuxmint.com](http://www.linuxmint.com/ "Linux Mint") site, select download and Linux Mint Debian, or click [here](http://www.linuxmint.com/download_lmde.php "Linux Mint Debian Download Page") to go directly. Under important links, select MATE/Cinnamon for Gnome 2 and 3 respectively, or Xfce for the Xfce Desktop. There are 32 or 64 bit options. In my case I chose **Gnome and 64 bit**.

[![LMDE download page](/old-blog/uploads/2012/05/linuxmint-download1.png "Downloading Linux Mint Debian Edition.")](/old-blog/uploads/2012/05/linuxmint-download1.png "Downloading Linux Mint Debian Edition.")

Downloading Linux Mint Debian Edition.

On the next page you can choose a closer mirror to download via http or torrent. I chose torrent to ensure the integrity of the download since torrent verifies the file's md5.

[![List of mirrors to download.](/old-blog/uploads/2012/05/linuxmint-download2.png "Choose a mirror to download.")](/old-blog/uploads/2012/05/linuxmint-download2.png "Choose a mirror to download.")

Choose a mirror to download.

With the iso file in hand, you can burn a CD or create a boot pendrive. My computer doesn't have an optical drive, so I'll need [Unetbootin](http://unetbootin.sourceforge.net/ "Unetbootin"). A utility to create boot pendrives for installing Linux. There are versions for windows, linux and mac.

[![Unetbootin download page](/old-blog/uploads/2012/05/unetbootin.png "Downloading Unetbootin, a boot pendrive creator.")](/old-blog/uploads/2012/05/unetbootin.png "Downloading Unetbootin, a boot pendrive creator.")

Downloading Unetbootin, a boot pendrive creator.

When you run Unetbootin, select "Disk image" ISO and, with the "…" button, navigate to the LMDE file you downloaded. Select the drive letter where the pendrive is inserted and click OK.  

It's important to already have the pendrive formatted in Fat32 beforehand so this step completes successfully.

[![Unetbootin](/old-blog/uploads/2012/05/unetbootin-linuxmint.png "Preparing the pendrive for installation.")](/old-blog/uploads/2012/05/unetbootin-linuxmint.png "Preparing the pendrive for installation.")

Preparing the pendrive for installation.

After creating the boot disk, insert it into your computer, enter the bios and select the boot order option for the correct pendrive or cd. This step is important and depends on each BIOS model, so it isn't described here.

When you turn on the computer with the boot disk inserted, the following screen will appear. Select **default** and press enter:

[![Grub menu](/old-blog/uploads/2012/05/20120523_153506-e1338267773228.jpg "Grub boot menu with UnetBootin")](/old-blog/uploads/2012/05/20120523_153506-e1338267801819.jpg "Grub boot menu with UnetBootin")

Grub boot menu with UnetBootin

  
The system will boot into the livecd and when it's ready, click the **Install Linux Mint** icon

[![Linux Mint Desktop](/old-blog/uploads/2012/05/Screenshot-e1338270772851.png "Install Linux Mint Debian Edition")](/old-blog/uploads/2012/05/Screenshot-e1338270772851.png "Install Linux Mint Debian Edition")

Install Linux Mint Debian Edition

  
Choose your language. I prefer Linux in English to make it easier to search for information on the internet, but you can choose whatever language you prefer.

[![Language selection](/old-blog/uploads/2012/05/Screenshot-1.png "Language selection for installation")](/old-blog/uploads/2012/05/Screenshot-1.png "Language selection for installation")

Language selection for installation

  
Choose your time zone:

[![List of time zones](/old-blog/uploads/2012/05/Screenshot-2.png "Choose your time zone.")](/old-blog/uploads/2012/05/Screenshot-2.png "Choose your time zone.")

Choose your time zone.

  
Choose your keyboard layout. Keyboards with a cedilla are Portuguese (Brazil), and without a cedilla you can choose: United States – English International.

[![Keyboard configuration](/old-blog/uploads/2012/05/Screenshot-3.png "Choose the keyboard layout.")](/old-blog/uploads/2012/05/Screenshot-3.png "Choose the keyboard layout.")

Choose the keyboard layout.

Select the HD where the system will be installed.

[![HD selection](/old-blog/uploads/2012/05/Screenshot-4.png "Choose the HD for installation")](/old-blog/uploads/2012/05/Screenshot-4.png "Choose the HD for installation")

Choose the HD for installation

Select the partition where Linux will be installed. In my case the disk is empty, so the partitions will need to be created. Click Edit partitions.

[![Partition list](/old-blog/uploads/2012/05/Screenshot-5.png "Selecting the partition for installation.")](/old-blog/uploads/2012/05/Screenshot-5.png "Selecting the partition for installation.")

Selecting the partition for installation.

The Gparted screen will open. Click Partition -> New.

[![Create Partition](/old-blog/uploads/2012/05/Screenshot-6.png "Creating the new partition.")](/old-blog/uploads/2012/05/Screenshot-6.png "Creating the new partition.")

Creating the new partition.

In the Free space following (MiB) field, enter the size in megabytes that will be left on the disk after creating the first partition. Basically, you leave the free space that you'll use for the swap partition. In my case I'll use the same amount as the memory I have. There's probably no need for that much, but when in doubt it won't hurt. In File system, select ext4 and click Add.

[![Partitioning the HD](/old-blog/uploads/2012/05/Screenshot-7.png "Leaving space for the swap partition.")](/old-blog/uploads/2012/05/Screenshot-7.png "Leaving space for the swap partition.")

Leaving space for the swap partition.

Select the unallocated space, go to the Partition -> New menu. We'll use the rest for the swap partition. In File system, select linux-swap. Click Add.

[![Swap partition](/old-blog/uploads/2012/05/Screenshot-8.png "Creating the Swap partition.")](/old-blog/uploads/2012/05/Screenshot-8.png "Creating the Swap partition.")

Creating the Swap partition.

Your disk will look like the figure below: Go to the Edit -> Apply All Operations menu.

[![Gparted after partitioning.](/old-blog/uploads/2012/05/Screenshot-9.png "Partitioning result.")](/old-blog/uploads/2012/05/Screenshot-9.png "Partitioning result.")

Partitioning result.

Confirm by clicking apply.

[![Confirming the changes.](/old-blog/uploads/2012/05/Screenshot-10.png "Confirming the changes.")](/old-blog/uploads/2012/05/Screenshot-10.png "Confirming the changes.")

Confirming the changes.

The partitions will be created. Close the Gparted window and click the refresh button. The created partitions will appear in the Linux installer. Right-click the ext4 partition and select Assign to /. The mount point of the primary partition will be /. Click forward.

[![Partition list and mount point.](/old-blog/uploads/2012/05/Screenshot-11.png "Setting the mount point.")](/old-blog/uploads/2012/05/Screenshot-11.png "Setting the mount point.")

Setting the mount point.

Enter the requested information and the computer's name on the network (hostname).

[![Form for username and password](/old-blog/uploads/2012/05/Screenshot-12.png "Username, password and hostname")](/old-blog/uploads/2012/05/Screenshot-12.png "Username, password and hostname")

Username, password and hostname.

On the next screen, leave the default or choose a specific partition for Grub.

[![Partition list for grub](/old-blog/uploads/2012/05/Screenshot-13.png "Where to install Grub?")](/old-blog/uploads/2012/05/Screenshot-13.png "Where to install Grub?")

Where to install Grub?

Review the settings and click install.

[![Reviewing the settings](/old-blog/uploads/2012/05/Screenshot-14.png "Reviewing the settings.")](/old-blog/uploads/2012/05/Screenshot-14.png "Reviewing the settings.")

Reviewing the settings.

The installation and file copying process will start. Go grab a coffee and come back in a few minutes.

[![Copying the files.](/old-blog/uploads/2012/05/Screenshot-15.png "Copying the files.")](/old-blog/uploads/2012/05/Screenshot-15.png "Copying the files.")

Copying the files.

When the installation finishes, click Quit and restart the computer. Don't forget to remove the pendrive.

[![Installation complete.](/old-blog/uploads/2012/05/Screenshot-16.png "Done!")](/old-blog/uploads/2012/05/Screenshot-16.png "Done!")

Done!
