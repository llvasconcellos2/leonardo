elFinder, a File Manager for WordPress

@@@
![elfinder logo](/old-blog/uploads/2012/06/elfinder.jpg "elfinder")

Once again a need arose while writing articles for this blog, and I decided to share it with you.

I use the [SyntaxHighlighter Evolved](http://wordpress.org/extend/plugins/syntaxhighlighter/ "SyntaxHighlighter Evolved") plugin to add color to the code posted on this blog, but I was having a problem with the theme I'm using. The theme sets the box-shadow attribute in the CSS for the <code> tag, which is also used by SyntaxHighlighter, and you can imagine the mess.

The fix is simple: edit the SyntaxHighlighter Evolved CSS and set box-shadow: none; for the <code> tag.

So far so good. No problem. It's just that it's a quick, simple change and I didn't want to have to open an ftp program, download the file, change it and upload it again. For small changes to files like this, it's quite handy to have a web file manager at hand.

Back in my Joomla days I used [eXtplorer](http://extplorer.sourceforge.net/ "eXtplorer"), a great file manager written in PHP with a frontend in [ExtJS](http://www.sencha.com/products/extjs/ "ExtJS"). We have a post dedicated exclusively to it:

As explained in the post, eXtplorer even has a plugin for WordPress, but it doesn't work on version 3.3.2.

Looking for more alternatives, I found a plugin called [wp-FileManager](http://wordpress.org/extend/plugins/wp-filemanager/ "wp-FileManager")which, just from looking at the screenshot, made me reject it. Forgive the prejudice, but my experience says that, with rare exceptions, programs with a "clunky" interface like this plugin's aren't updated often or are very amateurish.

[![wp-FileManager](/old-blog/uploads/2012/06/wp-filemanager1.jpg "wp-FileManager")](/old-blog/uploads/2012/06/wp-filemanager1.jpg "wp-FileManager, would you install it?")

wp-FileManager, would you install it?

elFinder  
After discarding wp-FileManager, without even installing it to test, I finally found a great replacement for eXtplorer. [elFinder](http://wordpress.org/extend/plugins/wp-elfinder/ "elFinder") even has practically the same _features_ as the other one, which are:  

  

*   Common operations on files stored on remote servers (copy, move, upload, create file/folder, rename, etc)
*   Two view types: list and icons
*   High performance on both the client and server side
*   Multiple file upload
*   Standard methods for grouping file selection using mouse or keyboard
*   Context menu
*   "Hot Keys" for most actions
*   Move/Copy files using Drag & Drop
*   Working with compressed files
*   Preview for most file types
*   Edit text files and resize images
*   "Places" for favorites
*   Directory size calculation
*   Thumbnail creation for image files running in the background
*   Easy integration with web editors (elRTE, CKEditor, TinyMCE)
*   Flexible configuration of access rights, allowed file types for upload, user interface, among others
*   Extensible
*   Simple JSON-based client/server API

  
I installed it on WordPress 3.3.2, despite a warning saying it hadn't been tested for this version yet, but everything worked well. I liked the plugin and I recommend it!
