To run the exploit that grants root access on the Motorola Droid Razr XT910's Android you'll need the following files / programs:

*   [Motorola Drivers](http://www.4shared.com/rar/_8dFBW7gba/MotorolaDeviceManager_234.html "Motorola Device Manager") (Motorola Device Manager)
*   [Oracle Virtualbox](https://www.virtualbox.org/wiki/Downloads "Oracle VirtualBox VM") ([Version 4.3.6 for Windows](http://www.4shared.com/file/TltYiF7Xba/VirtualBox-436-91406-Win.html "Oracle VirtualBox VM 4.3.6 for Windows"))
*   [Appliance](http://www.4shared.com/rar/bjlUlN_Yce/Root_Bionic_JB.html "Root Android for the Motorola Droid Razr XT910") for VirtualBox containing the root exploit

Once the downloads are done, let's start the procedure:

1.  Install the Motorola Device Manager.
2.  Install VirtualBox.
3.  Extract the Root\_Bionic\_JB.rar file.
4.  Open VirtualBox and go to **File – Import Appliance (Ctrl+I)**.
5.  Select the Root\_Bionic\_JB.ova file and click **Next (N)**.
6.  Don't change anything in the Appliance settings. Just click **Import**.
7.  Then go to **Settings – USB**, remove the listed devices and add your Razr (Motorola XT910), then click OK as shown in the image below:
    
    [![VirtualBox Motorola Droid Razr Root Appliance](/old-blog/uploads/2014/01/virtualbox_root_razr.jpg "VirtualBox Motorola Droid Razr Root Appliance")](/old-blog/uploads/2014/01/virtualbox_root_razr.jpg)
    
    Selecting the Razr in the VM's USB device list
    
8.  Start the VM by clicking **Start (T)**.
9.  The Razr and your computer must be on the same WIFI network. The VM will pause, showing a message asking you to connect the Razr to the computer: on the Razr, open the "Files" app and select **Remote storage – Add Storage**.
10.  In the host IP address field, enter the "Host IP address" shown in the VM. In my case: 192.168.0.13.
11.  Domain name: MYGROUP
12.  Shared folder name: share
13.  Username: guest
14.  Password: bionic
15.  Click connect on the Razr and then press Enter in the VM.
16.  The VM will show the message: CONFIRMED! Phone connected to SMB. If this message doesn't appear, check the information, username/password, and try again.
17.  Make sure your device has USB Debugging enabled (**Settings – Developer Options – USB Debugging**).
18.  Make sure your device is set to use the MTP protocol (**Settings – Storage – Menu button – USB computer connection – Media device (MTP)**).
19.  Press Enter in the VM.
20.  The exploit process will start. Your phone may look like it restarted; ignore it.
21.  Wait until the phone vibrates, then press Enter in the VM.
22.  It will show the message "Cleaning Up" and then ask you to press Enter. Press Enter in the VM.
23.  Your phone will restart and your Razr will already be rooted.
