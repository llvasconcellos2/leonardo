Greetings, earthlings!

This is the first Post of many that should come, or at least I hope so! 🙂

I decided to write this blog to record the difficulties and solutions I run into day to day involving technology. To be precise, I actually decided to write it to record my adventures in the Linux world.

I've known Linux since 1999, when I took a Conectiva Linux course (today [Mandriva](https://pt.wikipedia.org/wiki/Mandriva_Linux "Conectiva / Mandriva Linux")) at [Dualline](http://www.dualline.com.br/ "Dualline"), where I worked at the time. My teacher was even [Augusto Campos](https://plus.google.com/105003069076649186723/posts "Augusto Campos"), better known as Brain from the [br-linux.org](http://br-linux.org/ "br-linux.org") site. After the course I administered a web server running [Linux Slackware](http://www.slackware.com/ "Slackware") at SCTEL, a telecommunications company I worked at after leaving Dualline, and besides that I've done several server installations as a network infrastructure service provider. I usually installed RedHat or CentOS after it was released. Despite having contact with Linux throughout my professional life, until 2011 I hadn't used it on my personal machine. I started using [Ubuntu](http://www.ubuntu.com/ "Ubuntu") as my main system on a dual-boot machine with Windows 7 when I went to work with Java web development at [Totvs](http://www.totvs.com/ "Totvs") (yes, I've worked at five hundred different companies and that's a subject for another post). I made this decision after realizing that Java application servers like [JBoss](http://www.jboss.org/ "JBoss"), from RedHat, or [GlassFish](http://glassfish.java.net/ "Glassfish"), from Oracle, integrated better with Linux than with Windows. In fact, it's common for a Java Enterprise server to run on Linux.

So, my daily experience with Linux was with Ubuntu, right after the adoption of Unity as the default graphical interface. I don't know whether it was my fault, Unity's fault, or the fault of the millions of tweaks I made to the system, but my relationship with Ubuntu was always far from stable. So now I decided to use Linux Mint Debian, because I heard it's more stable and a bit friendlier than Debian itself. So what motivated me to write this blog is that every time I format my machine to install a different Linux distro, I make several adjustments to the system and install several programs, and then I forget how it's done :-P. This time I decided to write it down!

I can also share my experiences with Web and Mobile programming and some news on relevant topics, or anything else that comes to mind.

I hope you like the blog, so it serves as something more than just my notes.

Cheers,
